25-Point Security Hardening Checklist

✓ Never commit API keys to git
✓ Use read-only API keys where possible
✓ Enable 2FA on all connected accounts
✓ Review ClawHub skill permissions
✓ Separate Stripe account for agent products
✓ Environment variables for all secrets
✓ Audit installed skills monthly
✓ Review agent shell command history
✓ Set spending alerts on API accounts
✓ Use trash instead of rm
✓ Limit external message sending
✓ Keep OpenClaw updated
✓ Use VPN for remote access
✓ Review AGENTS.md boundaries
✓ Separate personal and agent file systems
✓ Encrypt sensitive files at rest
✓ Set up automated backups
✓ Use a password manager
✓ Audit connected accounts quarterly
✓ Monitor for unauthorized API usage
✓ Use rate limiting on agent actions
✓ Log all external actions
✓ Review and prune MEMORY.md
✓ Test disaster recovery
✓ Document your entire setup
